# go-etl

go-etl将提供的etl能力如下：

1. 主流数据库的数据抽取以及数据加载的能力，这个计划在storage包中实现
2. 类似datax的数据同步能力，这个计划在datax包中实现
3. 数据库间的数据校验能力，这个计划在libra包中实现
4. 以mysql sql语法为基础的数据筛选、转化能力，这个计划在transform包中实现（筹划中）
